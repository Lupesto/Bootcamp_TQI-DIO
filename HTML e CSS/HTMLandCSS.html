Sobre o HTML:

O Html foi criado em 91 por Tim Berners, a mais recente versão foi lançada em 2014.


Estrutura básica HTML: 


      <htmlp>, <headp>, <bodyp>, <metap>


Mudanças com a nova atualização:

Antes faziamos tudo com div, significando tudo com class, agora há novos elementos


Novos elementos:


      <sectionp>
      <asidep>
      <headerp>
      <articlep>
      <footerp>

Descrição dos elementos:

  • html -> todos elementos devem estar dentro de HTML [lang no htm]
  • head -> meta dados ficam nesta tag, ex charset UTF-8(codificação de caracteres)
  • script -> javaScript
  • style e link -> css
  • body -> fica conteúdo visível como textos, imagens e vídeos
  • section -> Representa uma seção genérica de conteúdo quando não houver um elemento mais específico para isso.
  • header -> cabeçalho da página ou de uma seção da página e normalmente contém logotipos, menus, campos de busca.
  • article -> Representa um conteúdo independente e de maior relevância dentro de uma página, como um post de blog, uma notícia em uma barra lateral ou um bloco de comentários. Um article pode conter outros elementos, como header, cabeçalhos, parágrafos e imagens.
  • aside -> É uma seção que engloba conteúdos relacionados ao conteúdo principal, como artigos relacionados, biografia do autor e publicidade. Normalmente são representadas como barras laterais.
  • footer -> Esse elemento representa o rodapé do conteúdo ou de parte dele,
